## 🌌 Overview

`nano-ai` is a lightweight and efficient AI adapter for integrating Gemini Nano AI model into WASM frameworks.

## 🚀 Features

| Method                  | Supported | Tested |
|-------------------------|-----------|--------|
| `get_capabilities`         | ✅        | ✅     |
| `create_session`         | ✅        | ✅     |
| `send_prompt`          | ✅        | ✅     |
| `stream_prompt`        | ✅        | ⬜     |
| `destroy_session`     | ✅        | ✅     |
